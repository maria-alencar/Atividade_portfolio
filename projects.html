Projeto 1

Portfólio Tech - Maria Fernanda

Seria um site pessoal para exibir meus projetos, habilidades e trajetória na área de tecnologia. O portfólio incluirá seções como 'Sobre Mim', 'Projetos', 'Habilidades' e 'Contato', com um design moderno e responsivo. Será desenvolvido usando HTML, CSS e JavaScript, e hospedado no GitHub Pages. O objetivo é criar uma plataforma que mostre meu crescimento como desenvolvedora e facilite conexões profissionais.


Projeto 2

Spotify Clone: Recriando a Experiência de Streaming Musical

Um projeto de recriação da interface do Spotify, focado em replicar a experiência de streaming musical com um design moderno e responsivo. Desenvolvido com HTML, CSS e JavaScript para o front-end, o projeto inclui funcionalidades como reprodução de músicas, criação de playlists e uma interface intuitiva para navegação. O objetivo é praticar habilidades de desenvolvimento web e oferecer uma alternativa personalizada à plataforma original.


Projeto 3

Galeria de Fotos Online

Uma galeria de fotos online onde você pode exibir e compartilhar suas melhores imagens de viagens, momentos especiais e projetos fotográficos. Desenvolvida com HTML, CSS e JavaScript para o front-end, e integração com um banco de dados Firebase para armazenamento, a plataforma permitirá o upload de fotos, organização em álbuns temáticos (como viagens, natureza, eventos) e compartilhamento privado ou público com amigos e familiares. O objetivo é criar um espaço personalizado para guardar e dividir suas memórias de forma fácil e acessível.
